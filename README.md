# MediaPipe Gesture Demo / 手势识别示例

## 中文说明

这是一个基于 MediaPipe 的手势识别示例项目，支持本地视频手势识别。

### 运行方式

```bash
python3 test-gesture-demo.py --source video/test.mp4
```
